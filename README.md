📊 Project Overview

This project showcases an interactive Sales Analysis Dashboard built using Microsoft Excel to analyze order data, revenue trends, and customer behavior.

The dashboard helps businesses track performance, identify trends, and make data-driven decisions using historical sales data.

🚀 Key Features

📈 KPI Metrics (At a Glance)
💰 Total Revenue: ₹35L+
📦 Total Orders: 1000+
⏱️ Average Delivery Time: 5.5 days
👤 Avg Customer Spend: ₹3,520+

📊 Sales & Trend Analysis
Monthly revenue trends using line charts
Identification of seasonal peaks and low-demand periods

🎁 Occasion-Based Insights
Revenue breakdown across occasions:
Diwali, Raksha Bandhan, Holi, Valentine’s Day, Birthdays
Helps understand event-driven sales behavior

🏆 Product Performance
Top 5 products contributing to revenue
Helps identify best-selling items
🌍 City-Level Analysis
Top 10 cities based on number of orders
Supports regional strategy planning

🎛️ Interactive Dashboard
Slicers for:
Occasion
Order Date
Delivery Date
Dynamic filtering for real-time analysis
🛠️ Tools & Technologies
Microsoft Excel
Power Query (Data Cleaning & Transformation)
Pivot Tables & Pivot Charts
Slicers & Timeline Filters
Advanced Excel Functions (XLOOKUP, IF)

📈 Key Insights
📊 Revenue Growth: Strong spikes observed during festive occasions
🎯 Customer Behavior: High average spend indicates strong premium buying patterns
🌍 Top Cities: Certain cities contribute significantly to total orders
🎁 Occasion Impact: Festivals and special days drive major revenue share
📦 Product Trends: A few products dominate overall sales
